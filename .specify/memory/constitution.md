# Constitution Pointer

This repo does not keep a local copy of the constitution. The governing constitution for every `traverse-framework` repo, including this one, lives in [`traverse-framework/.github`](https://github.com/traverse-framework/.github/blob/main/constitution.md).

This repo has adopted **governance version 1.0.0** (see that repo's `constitution.md` header for its version). Adopting a newer version is a deliberate step — update this pointer file's version note when this repo formally adopts a new governance version, don't let it drift silently.

Repo-specific product scope (what this repo's current milestone includes/excludes) lives in [`specs/001-registry-foundation/spec.md`](../../specs/001-registry-foundation/spec.md), not here.
